依赖关系：
GPU ==决定==》CUDA版本 
   CUDA版本 ==决定==》torch\torchvison\torchaudio 的版本（即pytorch版本) 
      pytorch版本 ==决定==》python版本
      pytorch版本 ==决定==》transformers版本

简言之：物理显卡 → 驱动 → CUDA → PyTorch → Python → Transformers
=================================================================================================================

# 0.创建python3.10
    conda create -n ml python=3.10
    conda activate ml 
# 1.清理现有环境
     conda activate ml 
     pip uninstall torch torchvision torchaudio -y  
     pip cache purge
# 2.安装cuda 12.8 Nightly版本。安装时会自动匹配相应的torch、torchvision、torchaudio版本(cu128表示12.8)
    pip3 install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu128
    安装完成后：
    Successfully installed torch-2.8.0.dev20250607+cu128 torchaudio-2.8.0.dev20250607+cu128 torchvision-0.23.0.dev20250607+cu128
